## 3. 英雄的旅程
```

# 每个游戏都要有英雄和目标。

# 使用game.spawnPlayerXY("captain", x, y)
# 为你的游戏添加一个英雄：
game.spawnHeroXY("captain", 20, 20)
# 使用game.addMoveGoalXY(x, y)
# 为你的游戏添加一个移动目标：
# 它需要离英雄10米远。
game.addMoveGoalXY(30, 20)
# 如果你想，你可以使用spawnXY("fence", x, y)
# 创建一个有围栏的迷宫…
game.spawnXY("fence", 20, 30)
# 然后，点击"测试关卡"来尝试你第一个可玩的游戏！

```
