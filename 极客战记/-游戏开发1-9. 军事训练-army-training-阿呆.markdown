## 9. 军事训练
```

# 生成一个英雄并添加一个目标。
game.spawnPlayerXY("champion", 40, 15)
game.addDefeatGoal()

# 生成至少2个矮人"munchkin"。
game.spawnXY("munchkin", 20, 25)
game.spawnXY("munchkin", 20, 25)
# 生成至少2个投矛手"thrower"。
game.spawnXY("thrower", 10, 10) # 食人魔投矛手
game.spawnXY("thrower", 10, 10) # 食人魔投矛手
# 生成至少2个士兵"soldier"。
game.spawnXY("soldier", 10, 10) # 人类士兵
game.spawnXY("soldier", 10, 10) # 人类士兵
# 生成至少2个弓箭手"archer"。
game.spawnXY("archer", 10, 10)  # 人类弓箭手
game.spawnXY("archer", 10, 10)  # 人类弓箭手

```
